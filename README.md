# 1º AVALIAÇÃO DE TOPICOS ESPECIAIS
## APLICATIVO GERADOR DE SENHAS!

## Como usar
- Coloque o tamanho que você deseja a senha
- O tamanho deve está entre 1 e 15
- Clique no botão de Gerar Senha
- Copie a senha gerada
